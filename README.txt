SDK version 1.4.5
Python version 3.7

This is a python wrapper for the bitbrain SDK.
The bbt.py module is the main entry point of for the python bindings.
It is self documented and contains a simple example.
Make sure that the python environment is in the correct version

The python bindings has almost a one to one relation with the C SDK, so you can look for further in the SDK documnetation in folder docs.

You can contact us at support@bitbrain.com for any issues you find.
